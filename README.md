## General info
This is a cars project where sellers post car ads and buyers can view those ads and contact sellers.

## Technologies
Project is created with:
* DRF
* MySQL
* Docker
* REST API
* Nginx
* Swagger

## Setup
To run this project: 

- add pipenv 
- write to terminal "docker compose up --build"
